# AppStream Metadata Creator

Графическое приложение для генерации XML-метаданных в формате AppStream.

#О приложении
AppStream Metadata Creator — приложение на Vala/GTK4/Libadwaita для создания метаданных, необходимых для публикации приложений в Flathub и других Linux-магазинах приложений (GNOME Software, KDE Discover).
Приложение предоставляет графический интерфейс для заполнения всех необходимых полей и автоматической генерации валидного XML-файла.

#Возможности
- Выбор типа компонента: desktop-application, console-application, addon, runtime
- Поддержка всех обязательных полей AppStream (ID, Name, Summary, License, Developer)
- Работа с составными полями: скриншоты, релизы, цвета брендинга
- Валидация данных: проверка URL, дат, обязательных полей
- Предпросмотр генерируемого XML
- Загрузка и сохранение файлов
- Интерфейс на Libadwaita с фиолетовой темой
- Модульная архитектура для легкого расширения
